# Pomodoro Timer App

Aplicación móvil desarrollada con Flutter y Dart que implementa la técnica de productividad **Pomodoro** para gestionar el tiempo de estudio o trabajo.

## Descripción
Esta aplicación permite a los usuarios trabajar en intervalos de tiempo enfocados (Pomodoros) seguidos de descansos cortos, ayudando a mejorar la concentración y productividad.

## Tecnologías utilizadas
- Flutter
- Dart

## Funcionalidades
- Temporizador de 25 minutos (Pomodoro)
- Pausas cortas entre sesiones
- Reinicio automático del temporizador
- Interfaz simple e intuitiva

## Instalación

Clonar el repositorio:

git clone https://github.com/Zttxw/Timer---Pomodoro---Dart.git

Entrar al proyecto:

cd pomodoro_timer

Instalar dependencias:

flutter pub get

## Ejecutar la aplicación

flutter run
